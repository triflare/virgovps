# Virgo VPS

With Virgo VPS, you can build your own virtual private server with Docker, without ripping your hair out.

## Key Features

- **Easy to use:** Virgo VPS is rather simple compared to other VPS solutions; just create your Docker container by setting a password with `VIRGO_PASS` and you're good to go.
- **Secure:** Docker's nature makes it so your Virgo VPS is completely seperate from your host system, so you don't have to worry about any security issues.
- **Customizable:** You can customize your Virgo VPS by installing any software you want, as long as it's compatible with your base image.
